# Task2-DataVis
## Objective
Create an interactive dashboard using Tableau Public to visualize Superstore sales data and communicate business insights.

## Tool Used
- Tableau Public

## Dataset
Sample Superstore

## Visualizations Created
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales by Segment
- Top 10 Products by Sales

## Key Insights
- Technology category generated the highest sales.
- Some regions were significantly more profitable than others.
- Monthly sales revealed seasonal patterns.
- Consumer segment accounted for the largest share of sales.
- The dashboard identified the top-performing products.
